📖 Guía de configuración
1
Crear proyecto Firebase
Ve a firebase.google.com → Ir a la consola → Agregar proyecto → ponle nombre → continuar hasta crear.
2
Activar Realtime Database
En el menú izquierdo: Compilación → Realtime Database → Crear base de datos → Ubicación: us-central1 → Modo de prueba → Habilitar.
3
Copiar la URL
Verás algo como: https://mi-proyecto-rtdb.firebaseio.com
Copia esa URL y pégala arriba en "Configuración".
4
Subir el JSON
Selecciona el archivo de respaldo de SuperCircuitoApp, verifica el preview y toca "Subir a Firebase".
5
Sincronizar en el reloj
Abre Samsung Internet en el Galaxy Watch 7 → navega a la URL de GitHub Pages → toca el botón ↻ SYNC en la esquina superior izquierda.
6
Para la próxima visita
Exporta nuevo JSON desde CircuitoApp → abre esta página en el celular → sube el nuevo archivo → toca SYNC en el reloj. ¡Listo!
